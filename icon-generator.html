How to Create Your App Icons:

  1. Open your browser's Developer Tools (F12 or Ctrl+Shift+I)
  2. Right-click on each icon below and select "Capture node screenshot" (Chrome) or use screenshot tools
  3. Save the screenshots with these names:
    • icon.png - 1024x1024 (main app icon)
    • adaptive-icon.png - 1024x1024 (same as icon.png)
    • favicon.png - 192x192 (web favicon)
  4. Save all images to assets/images/ folder
  5. Or use an online tool to convert this HTML to PNG at the correct sizes

Alternative: Use a screenshot tool like Snipping Tool (Windows) or take screenshots at exact dimensions.
